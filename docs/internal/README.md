# Internal Development Notes

These files are not the public product documentation. They exist to preserve
implementation notes, copy-work metadata, benchmark context, and local backlog
items that are useful for maintainers and coding agents.

## Files

- [copy-spec.md](./copy-spec.md) describes how runtime Russian copy should be
  generated and reviewed.
- [message-inventory.ru.yaml](./message-inventory.ru.yaml) is the metadata
  companion for copy work. Runtime strings stay in
  [`assets/copy/runtime.ru.yaml`](../../assets/copy/runtime.ru.yaml).
- [ingest-benchmark.md](./ingest-benchmark.md) documents the text and voice
  ingestion benchmark.
- [voice-cleaner-tuning.md](./voice-cleaner-tuning.md) records the current
  cleaner-only LLM tuning workflow.
- [backlog.md](./backlog.md) keeps local product backlog notes until they are
  promoted to GitHub issues.

Public setup and product positioning belong in the root
[`README.md`](../../README.md).
